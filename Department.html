IT Department Staff Structure

IT Manager/Director - Lerato Dhlamini

- Oversee the entire IT department and manage IT staff.

- Develop IT strategies aligned with business objectives.

- Budget planning and resource allocation for IT projects.

System Administrator - Kotla Gladwin Shadung

- Install, configure, and maintain operating systems (e.g., Windows, Linux).

- Manage server infrastructure (e.g., file servers, web servers, email servers).

- Ensure system security and apply patches and updates regularly.

Network Administrator - Lesego Matlhakola

- Install, configure, and maintain networking hardware (routers, switches, firewalls).

- Monitor network performance and security (e.g., bandwidth usage, packet loss).

- Troubleshoot network issues (e.g., connectivity problems, latency).

- Ensure proper configuration of VPNs, IP addresses, and DNS services.

Database Administrator - Xolani May

- Install, configure, and maintain databases (e.g., MySQL, SQL Server, Oracle).

- Ensure data integrity, security, and backup procedures.

- Monitor database performance and optimize queries.

- Design and implement database schemas and structures.

IT Support/Help Desk Technician - Mbali Khumalo

- Provide technical support to end-users for hardware, software, and networking issues.

- Troubleshoot and resolve technical problems remotely or in person.

- Assist with setting up and configuring new systems and software.

- Manage tickets using IT service management tools.

Cyber Security - Kutloano Motlhake

- Monitor network and system security to detect vulnerabilities and threats.

- Implement security controls (firewalls, encryption, authentication protocols).

- Conduct regular security audits and vulnerability assessments.

- Respond to security incidents (data breaches, hacking attempts).
