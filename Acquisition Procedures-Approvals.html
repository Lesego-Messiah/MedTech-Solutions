Acquisition Procedures/Approvals

The procedure for acquisition ensures that any new hardware or software added to the hospital's network is necessary, secure, and approved.

  1. Request Submission: Department heads submit a request to the IT department, specifying the need for new hardware or software.
  2. Evaluation: The IT department evaluates the request, ensuring compatibility with existing infrastructure and adherence to security protocols.
  3. Approval: The request is forwarded to the budget department and the IT manager for approval if deemed appropriate.
  4. Procurement: Once approved, the purchase is made from authorized suppliers, ensuring cost-effectiveness and compatibility with the hospital's systems.
